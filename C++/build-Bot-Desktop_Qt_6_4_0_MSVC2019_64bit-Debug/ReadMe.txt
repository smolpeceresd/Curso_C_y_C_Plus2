Le hago entrega del ejercicio del Bot de un concesionario. 

El ejercicio se basa en una Lista de Profundidad para la toma de decisiones. 

Para saber cosas del Concesionario es necesario introducir la palabra "concesionario"

Para saber cosas de los vehículos hay que jugar con las palabras permitidas en el arbol. 

En caso de no entender lo que se busca en 3 intentos, pasará a agente y cerrará la ejecución. 

Para salir del Flujo se debe escribir "Adios"
